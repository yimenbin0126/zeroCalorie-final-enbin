# <b>팀프로젝트 - 개인 기여 부분</b>
- 3조 아시겠조
- 프로젝트명 : 0칼로리
- 기능 목표 : 스프링을 활용한 게시판 구현
- 기술 스택 : Spring, Java, JSP, Ajax, Javascript, 외부라이브러리(jQuery), DBMS(Oracle), CSS
- 개발 환경 : Eclipse, JDK 11.0.15, Apache Tomcat 9.0, Maven
- 설명 : 다이어트 관리 웹 기반 애플리케이션 구현


## <b>맡은 역할</b>
- 6명 인원의 팀장
- 로그인, 회원가입, 아이디/비밀번호 찾기
- '고객센터' 주제의 CRUD 게시판
- 관리자용/회원모두용 게시판 CRUD, 게시판 답글, 좋아요/싫어요
- 조회수, 검색, 페이징, 파일 첨부기능, 댓글/대댓글 구현


## <b>제작 기간</b>
- 2022.10.02 ~ 2022.10.26
